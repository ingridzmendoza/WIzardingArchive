#Wizarding Archive

Wizarding Archive is a simple front-end project built to practice asynchronous JavaScript by consuming the public Harry Potter API.

Data is retrieved from:
https://hp-api.onrender.com/

Page is hosted in GitHub pages via:
https://ingridzmendoza.github.io/WIzardingArchive/

The project focuses on fetching, filtering, and displaying characters and spells using client-side API calls.
